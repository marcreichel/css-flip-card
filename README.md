# CSS Flip Card

A CSS hover flip card.


## Demo

[Live-Demo](https://marcreichel.github.io/css-flip-card/)


## Run Locally

Clone the project

```bash
  git clone https://github.com/marcreichel/css-flip-card.git
```

Go to the project directory

```bash
  cd css-flip-card
```

Open `src/index.html` in your browser.
